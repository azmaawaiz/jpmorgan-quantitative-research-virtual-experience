# Task 2 – Natural Gas Storage Contract Pricing

## Overview

This project develops a pricing model for natural gas storage contracts.

The model estimates contract profitability by considering injection and withdrawal prices, storage capacity, injection and withdrawal rates, and operational costs.

## Objectives

- Price a natural gas storage contract
- Calculate expected profit
- Incorporate operational constraints
- Evaluate storage economics

## Technologies Used

- Python
- Pandas
- NumPy

## Features

- Injection cost calculation
- Withdrawal revenue estimation
- Storage fee calculations
- Profit estimation
- Configurable contract parameters
