# Task 1 – Natural Gas Price Forecasting

## Overview

The objective of this project was to analyze historical natural gas prices and build a forecasting model capable of estimating future prices.

The model identifies historical trends and seasonal patterns in gas prices to provide future price predictions that can support financial and trading decisions.

## Objectives

- Analyze historical gas price data
- Identify long-term trends and seasonality
- Forecast future natural gas prices
- Visualize historical and predicted prices

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Features

- Data preprocessing
- Trend analysis
- Seasonal pattern detection
- Future price prediction
- Visualization of forecast results

## Files

- `JP Morgan Task 1.py` — Main forecasting script
- `gas_prices.csv` — Historical dataset
- `forecast_plot.png` — Forecast visualization

## Skills Demonstrated

- Time Series Analysis
- Data Cleaning
- Predictive Modeling
- Financial Data Analysis
- Data Visualization
