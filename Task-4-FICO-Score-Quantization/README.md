# Task 4 – FICO Score Quantization

## Overview

This project focuses on transforming continuous FICO credit scores into discrete risk buckets through score quantization.

The resulting buckets improve interpretability while maintaining predictive performance for credit risk assessment.

## Objectives

- Quantize continuous FICO scores
- Create interpretable credit score buckets
- Preserve predictive information
- Evaluate bucket performance

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features

- Score preprocessing
- Automatic bucket generation
- Quantization analysis
- Visualization of score distributions
