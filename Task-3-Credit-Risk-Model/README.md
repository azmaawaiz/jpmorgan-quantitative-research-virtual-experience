# Task 3 – Credit Risk Prediction

## Overview

The objective of this project was to develop a machine learning model capable of estimating the probability of borrower default.

Historical borrower information was used to train and evaluate a predictive model for credit risk assessment.

## Objectives

- Predict loan default probability
- Train a supervised machine learning model
- Evaluate model performance
- Analyze prediction accuracy

## Technologies Used

- Matplotlib

## Features
